# detect_face_by_yourself
opencv 
## 使用 OpenCV 首先安装环境 
1、建议使用anaconda 进行安装 


2、检测是否成功 使用第一个 detect函数 


3、采集脸部图片使用第二个take_pice 输入编号 


4、训练数据 如果这个时候出现错误，是系统的问题 ，重新安装下OpenCV 还要contrib 代码中有


5、进行预测，将你拍摄的照片对应的名字写入代码中的name中，依次对应。


6、运行predict函数就可以进行预测了。
